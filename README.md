## **Finlite — Simple User Flow**

### **1. Visiting the Finlite Website**

* The user opens the **Finlite homepage**.
* A landing page appears with the title:
  *“ Simple Bookkeeping Made Easy for Everyday Traders.”*
* There’s a clear **“Learn More”** button and a **“Login”** button.

  **Next Step:** User clicks **Learn More** to read about Finlite or **Login** to use the app

### **2. Learning About Finlite**

* Clicking **“Learn More”** opens a page that loads Finlite’s full **README.md** guide.
* It explains what the app does, how it works, and what problems it solves.
* Users can scroll through the document.

**Next Step:** User returns to homepage to sign up or log in.

### **3. Signing Up**

New users click **“Create an account”** to create a new account. They fill in:

  * Email
  * Password

After submitting, they’re taken to their **dashboard** where they can start using Finlite.

**Next Step:** User logs in to begin recording transactions.

### **4. Logging In**

* Returning users click **“Login”** and enter their email and password.
* Once logged in, they land on their **Dashboard**, where they can view and manage all business records.

**Next Step:** User accesses their dashboard to add sales or expenses.

### **5. Forgot Password (Change Password)**

* If a user forgets their password, they click **“Forgot Password?”**
* They receive a reset link via email.
* After setting a new password, they can log in again safely.

**Next Step:** User returns to login and continues.

### **6. Dashboard Overview**

The dashboard is simple and organized with clear buttons for every task:
* **Add Sales:** opens **Sales** to record the items sold and amount gotten. 
* **Add Expenses:** opens **Settings** to record purchases or running costs.
* **Calculate Profit:** Finlite instantly shows total profit or loss for the day.
* **View Report:** See clean summaries and charts of sales, expenses, and profits.
* **Download Report:** Save or print reports as PDF.

**Next Step:** User can record new data or review previous records

### **7. Settings**

The user can open **Settings** to:
  * Update their profile information.
  * Change password.
  * Adjust report format (e.g., daily, weekly, monthly).
  * Enable or disable notifications.

**Next Step:** Save settings and return to dashboard.

### **8. Getting Help or Tips**

The user can click **Help** or **Tips** icon from the menu. Finlite provides:

  * Step-by-step tutorials.
  * Short messages that explain how to use each feature.
  * Contact support if extra help is needed.
**Next Step:** User returns to use the app confidently.

### **9. Logging Out**

* When finished, the user clicks **Log Out**.
* They’re safely signed out and can log in again later.

##  **In Simple Terms**
Here’s the whole journey in plain flow:
1. **Open Finlite Website** → Read intro.
2. **Learn More** → View full README guide.
3. **Sign Up / Log In** → Access your dashboard.
4. **Forgot Password** → Reset password via email.
5. **Add Sales & Expenses** → Record daily transactions.
6. **View Profit or Loss** → See your business performance instantly.
7. **View Reports** → Generate clean reports, download as PDF.
8. **Change Settings** → Manage your account and preferences.
9. **Get Help / Tips** → Learn how to use features easily.
10. **Log Out** → End your session securely.

# Frequently Asked Questions (FAQ):

## General Questions 

- What is FinLite?
It is a simple, digital notebook app designed to help small business owners record sales and expenses quickly, see daily profit instantly, and export a clean report.
- Who is FinLite for?
It is for Market Traders, Small Shop Owners (salons, kiosks), POS Agents, and micro retailers who can't afford full accounting tools.
- What is the main benefit of FinLite?
To allow users to "Record sales and expenses in seconds. Know your profit before you sleep".

## Offline and Data
- Do I need the internet to use FinLite?
No, the app works fully offline (offline-first). It is perfect for markets and rural areas.
- Where is my data saved?
All data is saved locally using phone storage (SQLite).
- What if my phone gets damaged or I lose it?
The risk of data loss due to phone damage is mitigated by introducing optional cloud backup later.

## Usability and Security
- How do I know my profit?
The Dashboard (Daily Summary) displays your total sales, total expenses, and profits instantly.
- Is my information safe?
Yes, Finlite includes a Pin lock or biometric unlock for basic data privacy.
- How will low-tech users learn the app?
FinLite is designed to be visual with large buttons and minimal typing. Onboarding tutorial videos will be used, including some in Pidgin.
